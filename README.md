# Калькулятор рейтинга ЮУрГУ

Расширение для браузера, которое показывает текущий рейтинг по дисциплине в журнале ЮУрГУ.

## Что умеет

- считывает контрольные точки из раздела **«Текущий контроль»**;
- считает итоговый рейтинг с учётом весов работ;
- позволяет вручную изменить баллы прямо в окне расширения;
- позволяет добавить бонус в процентах;
- поддерживает светлую и тёмную тему.

## Как считается рейтинг

Для каждой контрольной точки считается процент выполнения:

```text
процент = набранные баллы / максимум баллов * 100
```

Затем проценты усредняются с учётом веса:

```text
рейтинг = сумма(процент работы * вес работы) / сумма(весов)
```

Бонус из поля **«Бонус (%)»** прибавляется к получившемуся рейтингу вручную.

## Какие строки попадают в расчёт

В расчёт попадают только строки из раздела **«Текущий контроль»**, у которых одновременно есть:

1. баллы в формате `получено / максимум`;
2. положительный числовой вес.

Строки без веса, с весом `-`, с нулевым весом или без нормального значения баллов автоматически игнорируются.

## Установка из GitHub

1. Скачайте актуальную версию расширения с GitHub: **Code → Download ZIP**.
2. Распакуйте архив в удобную папку.
3. Откройте страницу расширений браузера:
   - Chrome / Edge / Яндекс Браузер: `chrome://extensions`;
4. Включите **режим разработчика**.
5. Нажмите **«Загрузить распакованное расширение»**.
6. Выберите папку с расширением.

После обновления файлов с GitHub на странице расширений нажмите **«Обновить»**.

## Как пользоваться

1. Откройте журнал на `online.susu.ru`.
2. Выберите нужную дисциплину.
3. Нажмите на иконку расширения.
4. При необходимости измените баллы или добавьте бонус.

## Почему расширение не опубликовано в магазине

Публикация расширений в магазине браузера может требовать оплаты или дополнительных проверок. Поэтому актуальная версия распространяется через GitHub.

## Важно

Расширение работает только с текущей разметкой журнала ЮУрГУ. Если сайт изменит HTML-классы или структуру таблицы, парсер может потребовать обновления.
